Risque d'Accident

Cette application est mon projet final de la formation fullstack en data science. J'ai travaillé sur ce projet pendant deux semaines.

Après avoir décidé de créer une application permettant de prédire le risque d'accidant de trafic, j'ai commencé par chercher, exploirer et nettoyer les données.

Le système intégré dans l'application permet d'interpréter les données et de voir les endroits où ont lieu les accidents de trafic grâce à l'api google maps.

Le résultat final de prédiction est obtenu grâce aux models de machine learning .

Technologies:
  • - Python
  • - Pandas
  • - Numpy
  • - Scikit-learn
  • - Seaborn
  • - Matplotlib
  • - Logistic Regression
  • - Gradient Boosting Classifier
  • - ANN
  • - Google Maps Api
  • - Streamlit
  • - Heroku
