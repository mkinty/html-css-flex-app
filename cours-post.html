Site de cours en ligne de KmyProd

J'ai commencé à développer ce site en avril 2020. C'est mon premier projet de developpement web avec le framework django.

Le système d'administration est fait pour faciliter la tache aux instructeurs. Ils peuvent intégrer des fichiers, des images ou des vidéos dans le contenu de leurs cours.

Seuls les instructeurs ou personnes ayant droits peuvent modifier le contenu ou créer un cours.

Pour voir le contenu d'un cours, les utilisateurs doivent obligatoirement s'inscrire.

Technologies:
  • - Python
  • - Django + Django REST Framework
  • - Ajax
  • - JavaScript
  • - Postgres
  • - Cloud (digitalocean)
  • - Gunicorn
  • - Nginx
